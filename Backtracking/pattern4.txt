PATTERN 4 — Partition / Cut Problems
Recognition Clue:

“Split string”
“Partition into valid parts”
Try every possible cut

Examples:

Palindrome Partitioning
Restore IP Address
Tree Thinking:

At index → try every possible substring.

Core Template:
void solve(int start, List<String> ds) {

    if (start == s.length()) {
        ans.add(new ArrayList<>(ds));
        return;
    }

    for (int end = start; end < s.length(); end++) {

        if (isValid(start, end)) {

            ds.add(s.substring(start, end + 1));

            solve(end + 1, ds);

            ds.remove(ds.size() - 1);
        }
    }
}
